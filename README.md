# Shikor — Central Asian Hunts

Сайт охотничьего клуба Shikor: трофейная охота на границе Узбекистана и Казахстана.

## Технологии

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- i18next (ru / en / uz)

## Запуск

```sh
npm i
npm run dev
```

Сборка: `npm run build`

## Языки

По умолчанию — русский. Поддерживаются английский и узбекский (переключатель в шапке).
